Här lägger vi in din SuperSafe-inspelare. Tills dess är detta en placeholder.

Vill du bädda in din nuvarande recorder-sida?

Sätt din befintliga URL här så gör vi en enkel redirect:

// Byt till din nuvarande recorder-URL om du kör den på annan host:
const EXTERNAL_RECORDER_URL = '' // t.ex. 'https://ditt-projekt.onrender.com/recorder.html';

if (EXTERNAL_RECORDER_URL) {
  location.replace(EXTERNAL_RECORDER_URL);
}
